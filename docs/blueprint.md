# **App Name**: ClarityFlow

## Core Features:

- Financial Foundation Setup: Define personal costs, business expenses, and work schedule to calculate a True Minimum Hourly Rate (TMHR).
- Service Library & Profitability: Create a service menu, define durations, and link products to automatically calculate total cost and break-even point, use a 'Profit Tester' slider to adjust price.
- Weekly Planner: A seven-day calendar to log appointments, sales, and expenses to calculate the net profit.
- Inventory Hub: Manage all professional products, retail items, and equipment to calculate the cost-per-use.
- Client Log (CRM): Maintain a client database, view appointment history, track lifetime value, and keep notes.
- Public Booking Page & Client Portal: Generates a unique booking page for service browsing and appointment requests. Returning clients use the client portal to view history and manage their information.
- AI-Powered Insights: Chat with an AI 'CFO' tool for financial questions about your business performance. Receive an end-of-day debrief summarizing profits, losses, and inventory needs.

## Style Guidelines:

- Primary color: Deep violet (#7955c4), chosen to subtly invoke the historical association of purple with prosperity and wealth, while avoiding associations with royalty. In this app, purple is used to establish a visual connection with money and financial matters. Because purple can sometimes read as cold, we avoid bright, pure shades. In its place we embrace a tertiary hue of purple, leaning toward the warmer tint of magenta.
- Background color: Light grey (#f0f0f5), almost the same hue as the primary but extremely desaturated to be unobtrusive.
- Accent color: Teal (#4db6ac), approximately 30 degrees 'left' of the primary color. Slightly desaturated and significantly brighter to be vivid without clashing.
- Body and headline font: 'Inter', sans-serif.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and professional icons for navigation and features.
- Subtle transitions and animations for a smooth user experience.
