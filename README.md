
# ClarityFlow Studio OS

A high-fidelity **Business Orchestration Platform** built specifically for solo service professionals and high-performance studio teams. ClarityFlow replaces disconnected booking tools with a unified, yield-aware studio manifest.

## Strategic Pillars

- **Yield Architecture**: Automated **Total Minimum Hourly Rate (TMHR)** foundation. Every service is audited against studio overhead and burdened labor costs to ensure net profit protection.
- **Dossier Intelligence**: Comprehensive client records tracking **Lifetime Value (LTV)**, technical service formulas, and medical/compliance alerts.
- **The Terminal (POS)**: A high-fidelity point-of-sale built for group checkouts, retail liquidation, and automated membership perk enforcement.
- **Kiosk Mode**: An autonomous client check-in terminal that identifies returning guests and facilitates digital signature protocol (Consent/Intake).
- **Pro Team Hub**: Performance metrics, automated turn-order assignment (Fair Play vs. Manual), and unified payroll ledger integration.
- **AI CFO**: Strategic end-of-day debriefs and financial forecasting powered by Genkit AI.

## Technical Matrix

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & ShadCN UI
- **Database**: Firestore (Real-time synchronization)
- **Authentication**: Firebase Auth
- **AI Engine**: Genkit (Google Gemini 2.5 Flash)
- **State Management**: React Context + Real-time Listeners

## Compliance & Security

- **Strategic Intake**: Encrypted digital signature vault for consent and intake forms.
- **Payment Vault**: PCI-compliant card-on-file tokenization for session deposits and arrears recovery.
- **Audit Ledger**: Immutable transaction logs with digital receipt attachments.
